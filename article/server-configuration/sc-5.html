# 利用 DNSPod 管理域名解析

  • 域名需要域名解析服务器，需要 DNS 将域名解析到某个 IP 地址，通常 DNS 是购买域名服务商提供的；
  • 一个域名只可以对应一个 IP 地址，一个 IP 地址可以对应多个域名；
  • 域名放 DNSpod(dnspod.cn)批量管理；
