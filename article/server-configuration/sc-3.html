# 搭建 Nodejs 生产环境

安装 node（参考 Linux/安装最新 nodeJS.md）

配置国内淘宝源：

npm --registry=https://registry.npm.taobao.org install -g npm

增加系统文件监控数：

echo fs.inotify.max_user_watches=524288 | sudo tee -a /etc/sysctl.conf && sudo sysctl -p

测试：

<!--vi app.js-->
const http = require('http');
http.createServer((req, res) => {
    res.statusCode = 200
    res.setHeader('Content-Type', 'text/plain')
    res.end('Hello，world！')
})
    .listen(8081, () => {
        console.log('Server Running At 8081');
    })

# 借助 pm2 让 Nodejs 服务常驻

pm2 ：nodeJS 部署与进程管理工具，可以使 node 部署的服务器长期运行（出现异常可以自动重启）；

npm i pm2 -g

指定某程序长期运行：

pm2 start app.js

列出服务器上当前所运行的 node 服务：

pm2 list

看某一个应用(如 app.js)的详细信息：

pm2 show app

查看实时日志导航（退出 Ctrl + C）：

pm2 logs
