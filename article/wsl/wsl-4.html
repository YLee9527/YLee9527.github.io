# 美化 wsl

# Powershell 美化: oh-my-posh

Set-ExecutionPolicy RemoteSigned -scope CurrentUser

// 安装scoop
iex (new-object net.webclient).downloadstring('https://get.scoop.sh')

// 安装需要的软件
scoop install concfg     //如果需要卸载可以输入scoop uninstall concfg。

scoop install git
//此时，git的环境变量是设置在当前用户下的。如果需要设置在系统下，可以通过以下语句安装。需要使用sudo来提升权限，是的，就是Linux中的那个sudo。
scoop install sudo
sudo scoop install -g git

// 安装posh-git
scoop bucket add extras
scoop install posh-git

// 安装oh-my-posh
scoop install oh-my-posh

添加字体

统中默认的字体缺失了一些 oh-my-posh 需要用到的字符，所以需要安装一个可以支持的字体。

我选择 sarasa gothic 系列字体，可以在 sarasa gothic 下载，然后右键安装字体即可。

修改注册表

为了能够在配置里使用，还需要修改注册表。

按 WIN+R 打开运行，输入 regedit 打开注册表编辑器。

在

HKEY_LOCAL_MACHINE >> SOFTWARE >> Microsoft >> Windows NT >> CurrentVersion >> Console >> TrueTypeFont
1

路径下新增一个字符串项目，名称任意，内容为 Sarasa Term SC。

启用 oh-my-posh

敲$profile 可以让 PowerShell 告诉我们这个文件的路径是什么。
