# 枚举

使用枚举可以给一些难以理解的常量赋予一组具有含义的直观的名字，可以理解枚举就是一个字典,使用 enum 关键字定义，支持数字和字符串两种

# 数字枚举

// 数字枚举
enum Status {
  Uploading,
  Success,
  Failed
}
console.log(Status.Uploading) // 0
console.log(Status.Success) // 1
console.log(Status.Failed) // 2
// 可以从任意位置指定，后面一次递增
enum Status2 {
  Uploading,
  Success = 3,
  Failed
}
console.log(Status2.Uploading) // 0
console.log(Status2.Success) // 3
console.log(Status2.Failed) // 4
// 也可以每个都指定，不一定按顺序
// 注意：数字枚举在定义值的时候，可以使用计算值和常量，不过面紧接着的必须设置初始值
const text = 3 // 使用函数也同理
enum Status3 {
  Uploading,
  Success = text,
  //   Failed,  // error: 枚举成员必须具有初始化表达式。
  Failed = 5
}

// 反向映射
console.log(Status[0]) // Uploading
console.log(Status) // {0: "Uploading", 1: "Success", 2: "Failed", Uploading: 0, Success: 1, Failed: 2}

# 字符串枚举

// 字符串枚举
enum Message {
  Error = 'Sorry, error',
  Success = 'Hoho, success',
  Failed = Error
}
console.log(Message.Success) // Hoho, success
console.log(Message.Failed) // Sorry, error

# 异构枚举

即包含数字的值又包含字符串的值,(除非需求，一般不建议使用)

// 异构枚举： 即包含数字的值又包含字符串的值,(除非需求，一般不建议使用)
enum Result {
  Failed = 0,
  Success = 'success'
}

# 枚举成员类型和联合枚举类型

当一个枚举值满足一定条件的时候，那么这个枚举的每个成员和枚举值本身都可以作为类型来使用

满足条件：

  1. 不带初始值的枚举成员 enum E{ A }
  2. 值为字符串字面量 enum E { A = 'a' }
  3. 值为数值字面量 enum E{ A = -1 }
// 枚举成员类型：
enum Animals {
  Dog,
  Cat
}
interface Dog {
  type: Animals.Dog
}
const dog: Dog = {
  //   type: Animals.Cat, // 不能将类型“Animals.Cat”分配给类型“Animals.Dog”
  type: Animals.Dog
}
// 联合枚举类型：string | number
enum Turn {
  Off,
  On
}
interface Light {
  status: Turn
}
const light: Light = {
  status: Turn.Off // 或者Turn.On
  //   status: Animals.Cat, // errir: 不能将类型“Animals.Cat”分配给类型“Turn”
}

# 运行时的枚举

一个枚举值编译完之后就是一个真实的对象，所以可以在代码运行时的逻辑中使用枚举值，但是在接口这种定义了类型的，在编译之后是不能改变的

// const enum: 会直接编译成指定的值和不是枚举对象中的某个
const enum Animals1 {
  Dog = 1
}
console.log(Animals1.Dog) // 编译 => 1
console.log(Animals.Dog) // 编译 => Animals.Dog
